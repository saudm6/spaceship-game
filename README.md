# SpaceShip game

Install pygame on your terminal
# Windows
```bash
pip install pygame
```
# Mac
```bash
pip3 install pygame
```
# Controls for Yellow Spaceship (Left side)
W = Up <br>
A = Left <br>
S = Down <br>
D = Right <br>
Space = Shoot <br>

# Controls for Red Spaceship (Red side)
Up (arrow) = Up <br>
Left (arrow)= Left <br>
Down (arrow)= Down <br>
Right (arrow)= Right <br>
Right Control = Shoot <br>

Have Fun !!!
